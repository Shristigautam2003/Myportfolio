# Shristi Gautam — Portfolio Website

A fully responsive, dark-themed personal portfolio website.

## 📁 Folder Structure
```
shristi-portfolio/
└── index.html       # Complete single-file portfolio (all CSS, JS, image embedded)
└── README.md        # This file
```

## 🚀 Run Locally
Simply open `index.html` in any browser — no server needed.

## 🌐 Deploy to Netlify (Free, Easiest)
1. Go to https://netlify.com and sign up
2. Drag & drop the `shristi-portfolio` folder onto the Netlify dashboard
3. Your site is live instantly with a free URL!

## 🌐 Deploy to Vercel
1. Install Vercel CLI: `npm install -g vercel`
2. Inside the folder, run: `vercel`
3. Follow the prompts — your site gets a `.vercel.app` URL

## 🌐 Deploy to GitHub Pages
1. Create a new GitHub repo
2. Upload `index.html` to the repo root
3. Go to Settings → Pages → Source: main branch
4. Your site will be at `https://yourusername.github.io/repo-name`

## ✏️ How to Update
- **Profile photo**: Replace the base64 image in the `<img>` tags
- **Text content**: Edit directly in the HTML sections
- **Colors**: Change CSS variables at the top of the `<style>` block
- **Add projects**: Copy a `.project-card` div and update content
